# `widgets/` — Complex UI Sections

## Contains:

Large UI compositions that combine `features` and `entities`

Examples: **Header**, **Sidebar**, **ProductList**, **Navbar**

- Depends on: `shared`, `entities`, `features`
- Depended on by: `pages`
